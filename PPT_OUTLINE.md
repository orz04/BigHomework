# Warframe Wiki 微信小程序项目介绍

## 一、项目概述

### 1.1 项目背景
- Warframe 是一款热门的第三人称射击游戏
- 游戏内容丰富，数据量大，玩家需要便捷的查询工具
- 微信小程序具有轻量、便捷的特点，适合游戏攻略查询

### 1.2 项目目标
- 为 Warframe 玩家提供便捷的游戏数据查询服务
- 涵盖战甲、武器、模组、遗物等核心游戏内容
- 提供清晰的数据展示和筛选功能

---

## 二、技术架构

### 2.1 项目结构
```
├── components/          # 公共组件
│   ├── search-bar/      # 搜索栏组件
│   └── section-card/    # 卡片组件
├── data/                # 数据文件
│   ├── indexs.js        # 战甲数据
│   ├── weapons.js       # 武器数据
│   ├── mods.js          # 模组数据
│   ├── relics.js        # 遗物数据
│   └── ...
├── pages/               # 页面文件
│   ├── home/            # 首页
│   ├── indexs/          # 战甲列表页
│   ├── index-detail/    # 战甲详情页
│   └── ...
├── utils/               # 工具函数
├── app.js               # 应用入口
├── app.json             # 应用配置
└── app.wxss             # 全局样式
```

### 2.2 技术栈
- **框架**: 微信小程序原生框架
- **语言**: JavaScript (ES6+)
- **样式**: WXSS (微信小程序样式)
- **数据**: JSON 本地数据

### 2.3 核心特性
- 响应式设计，适配不同屏幕尺寸
- 数据驱动的页面渲染
- 模块化组件开发
- 本地数据缓存，提升加载速度

## 三、功能模块

### 3.1 首页模块
- 快速导航入口
- 搜索功能
- 热门内容展示

### 3.2 战甲系统
- 64个战甲数据展示
- 详细属性面板（生命值、护盾、护甲、能量等）
- 锻造需求展示
- 入手方式说明

### 3.3 武器系统
- 主武器、副武器、近战武器分类
- 武器属性对比
- 伤害类型展示

### 3.4 模组系统
- 模组列表和详情
- 极性标识
- 等级需求

### 3.5 遗物系统
- 古纪、前纪、中纪、后纪遗物分类
- 遗物奖励查询
- 虚空裂缝来源表

### 3.6 资源系统
- 各类资源展示
- 获取途径说明
